# README

URL shortener built with Rails + React

Visit the app at https://www.tny.fit/

## Prerequisites

ruby 3.0.0

node > 14.8.0

## Running the app

`rails s`

then go to http://localhost:3000/ on your browser

## Database creation

`rails db:create && rails db:migrate`

## How to run the test suite

`rspec ./spec`

## Deployment

Heroku with custom domain
